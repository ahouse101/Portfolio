Lab Spend

https://www.labspend.com/

Lab Spend maintains a comprehensive suite of tools targeting research labs. I've worked with them as a full stack developer since January 2016. The current site is build on a Laravel-based PHP stack with a responsive front-end using the Materialize CSS framework and Backbone.js.

In the summer of 2016, I led the port of the core website from its original plain PHP implementation with various server-side and client-side libraries to a new, streamlined architecture. We chose to adapt the Laravel framework to the project's needs, serving partially constructed Blade views as well as exposing a REST API that a reactive Backbone-based front-end could consume. This resulted in a hybrid Laravel website with each page existing as a compartmentalized single-page JavaScript application, eliminating several server side dependencies and homogenizing our various services' codebases.

UM Supermileage

http://umsm.engin.umich.edu/

University of Michigan's Supermileage team is a small team of student engineers that designs, builds, and competes with a highly efficient vehicle, creating a new design every year.

The original Supermileage website was a very old WordPress design. The team didn't have a web designer, so no real design work had gone into it. In 2015, I joined the team and started building a new website. Because it was important that other members of the team were able to upload photos and edit copy, I built upon the WordPress base with an entirely new template, as well as a streamlined Admin Dashboard. The team continues to manage the content year-by-year.
